Project 1

Portfolio and Resume Project

The first of my projects is my resume/ portfolio builder. This project is simple, it takes the things I have learned in the programming space and displays them for the world to see. This project is mostly HTML and JavaScript created in notepad and the inspect feature in my browser. This is not to make it harder on myself, but to help me actually learn how it all ties together outside of an IDE that auto corrects much of my work. This approach has led to some self-discovery, expansion on my skills and helped me get much more consistent in my work.

Experimentation and a desire to learn more has resulted in the code here. Not always the cleanest or best way and definitely still in flux. However, it does provide an example of my front-end abilities at the current time. Unfortunately, as my experience and knowledge grows, I learn new fun tricks and want to learn and experiment with them. However as the code has becomes more complicated I am needing to learn the backend as well.

Most of this code can be found in a text book, YouTube video, or even on StackOverflow. The code is not knew but the application is different than I have found in many cases.
